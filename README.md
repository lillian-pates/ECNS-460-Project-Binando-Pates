## ECNS 460 Semester Project 

### Objective
This project aims to utilize descriptive and spatial analysis to understand flood exposure in Poza Rica, Veracruz, México. This project serves as an extension to a parametric insurance capstone project. By examining the relationship between flood distance, exposure, and other spatial components, the estimate of economic impact will be improved, specifically in locations with limited data. These findings will then be used to create an effective exposure index that will be used to inform the design of a parametric insurance trigger.

See Topic Description and Data Information for additional project information.

**Research Question:**
How does flood exposure vary across elevation, river proximity, and floodplain extent in Poza Rica?
