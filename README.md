# ECNS-460-Project-Binando-Pates
ECNS 460 Semester Project 

This project aims to utilize predictive and spatial analysis to estimate the economic losses of a flooding event using a proxy, serving as an extension to a parametric capstone project. By examining the relationship between flood distance, exposure, and other spatial components to the proxy, the estimate of economic damages will be improved, specifically in locations with limited data. These findings will then be used to inform the design of a parametric insurance trigger.

See Topic Description, Data Information, and Tentative Analysis Plan for additional project information.
