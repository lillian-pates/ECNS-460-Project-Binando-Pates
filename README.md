## ECNS 460 Semester Project 

This project aims to utilize descriptive and spatial analysis to estimate the effective exposure of a flooding event, serving as an extension to a parametric insurance capstone project. By examining the relationship between flood distance, exposure, and other spatial components to the proxy, the estimate of economic damages will be improved, specifically in locations with limited data. These findings will then be used to inform the design of a parametric insurance trigger.

See Topic Description, Data Information, and Tentative Analysis Plan for additional project information.
