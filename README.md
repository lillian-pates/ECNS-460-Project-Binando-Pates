## ECNS 460 Semester Project 

### Objective
This project aims to utilize descriptive and spatial analysis to understand flood exposure in Poza Rica, Veracruz, México. This project serves as an extension to a parametric insurance capstone project. By examining the relationship between flood distance, exposure, and other spatial components, the estimation of economic impact will be improved, specifically in locations with limited data. These findings will then be used to create an effective exposure index that will be used to inform the design of a parametric insurance trigger.

See Topic Description and Data Information within references folder for additional project information.

**Research Question:**
How does flood exposure vary across elevation, river proximity, and floodplain extent in Poza Rica?

### Data Sources
- NASA Digital Elevation Maps
- Microsoft Building Footprints
- WorldPop: México - Spatial Distribution of Population
- DesInventar & News Source Flood Events
- LISFLOOD River Flow Forecast

### Repository Structure
- `data/`: raw and processed datasets
- `scripts/`: numbered data processing and modeling pipeline
- `outputs/`: figures, tables, and html files
- `references/`: project sources and additional information

### Reproducibility
1. Install required R packages
2. Run scripts in numerical order
3. Processed datasets will be saved to `data/processed`
4. Final outputs (figures, htmls) are saved in `outputs/`
